# npml
Npml(Numpy machine learning)，是一个基于numpy实现常见的机器学习算法的python包。

> 配套算法理论知识详解 👉 [https://github.com/xiligey/npml_theories](https://github.com/xiligey/npml_theories)

## 项目结构
- data: 存放一些测试用到的数据
- docs: 各个算法的理论介绍
- npml: npml的所有代码
- tests: npml单元测试目录
